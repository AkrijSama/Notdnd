## 1. THE PIVOT

The current active direction. It supersedes the prior "MVP" plan.

- **[LOCKED] Proprietary IP layer.** Keep 5e's uncopyrightable d20 math / resolver /
  coherence spine; replace WotC-flavored **content** with a proprietary rulebook
  (stats, races, items, weapons, backgrounds, crafting, laws) expressed as **data the
  engine reads**, expansion-book-ready (Naruto-5e homebrew model — the engine is the
  system, the rulebook is swappable content).
  - **Legal footing:** clean-room expression + one CC-BY-4.0 SRD 5.1 attribution
    line in the credits.
  - **Identity:** the standalone **"Inkborne System"** — zero 5e branding anywhere
    outside the legal credits.

- **[LOCKED] Universal stat spine.** `STR / DEX / VIT / Spirit / INT / Luck`.
  Derived stats (Attack, Evasion, Accuracy, turn-speed, Crit) compute from these.

- **[LOCKED] Turn economy = CTB.** FFX-style **discrete turn queue**: DEX/speed
  drives turn **frequency** via a server-computed deterministic tick queue;
  haste/slow/stun bend the queue. This **supersedes D.4's d20 initiative** (a partial
  D.4 rework is accepted cost).
  - **ATB rejected** — real-time, incompatible with a text medium.
  - **5e initiative rejected** — the exact feel being escaped.

- **[LOCKED] Resolution law.**
  - **No-stakes-no-roll:** automatic / check / contested tiers; observation and safe
    conversation never roll.
  - **Three-band resolution:** success / success-at-a-cost / failure-with-consequence.
  - **Skill list deleted:** raw stat checks + proficiency granted through
    world-defined **training domains**.

- **[LOCKED] Race contract.** Race = a stat boost + a feat. Race **content** is
  per-world; the race **contract** is universal.

- **[LOCKED] Rulebook Chapters 7 / 8 / 9 — Progression, Life & Death, Items &
  Crafting.** Confirmed by director ruling 2026-07-05: committed-event XP; FF
  growth-profiles as data; level cap 20; universal death-save machinery with
  return-from-death as world lore; the bounded ten-status FF-canon vocabulary;
  item = slot + stat-mods + one ability, with Weapon / Armor / Accessory×2 slots;
  crafting as a contract-only schema. All numeric defaults in the written
  chapters approved as defaults (rarity budgets kept deliberately);
  **nat-20/nat-1 special rules on checks explicitly rejected** — struck from
  Ch3, which now codifies that the die's face has no special meaning on a check.

- **[LOCKED] Build sequence (strict order):**
  1. **Rulebook / Player's Handbook**, chapter by chapter, with the **Isekai world
     as the first reskin target**.
  2. **"Law of Creating Worlds" spec** — the valid-world schema (factions, figures,
     artifacts, power-systems, background) + UGC submission rules.
  3. **Author 3 worlds** — Isekai-tower, Cyberpunk, and the director's post-apoc IP.
  4. **Full onboarding REWORK** — turn-1 in-fiction lore-dump via a quest-giver,
     NPC-driven middle, an OOC channel, and a "paintbrush not hammer" GM.

- **Main manual = world-agnostic chassis only** (10 lean chapters). Races, classes,
  power-systems, bestiary, and lore live in per-world books.
